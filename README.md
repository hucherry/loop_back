# loopback

#### 项目介绍

快速回测期货策略的回测平台， 但是操作复杂点，需要将策略放在 strategy 里面来编译运行。
基于c++ 写的， 仿照原vnpy架构做的

速度基本是  市面上  tb(交易开拓者), multicharts 这些平台的十倍以上。   部分环节做了特殊优化
